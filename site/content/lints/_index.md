# Lints

Every lint ahklint ships, grouped by category. Each page is generated from the
lint's `meta` plus its hand-written documentation; the examples on each page are
the same fixtures the test suite runs, so the docs cannot drift from behavior.

The pages under this section are generated by `build/compile-docs.ahk` and are
not committed to the repository — this index page is the only hand-written file
in `content/lints/`.
